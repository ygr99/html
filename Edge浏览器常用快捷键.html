Edge浏览器常用快捷键

原文： Edge浏览器常用快捷键 -入门向导

功能 快捷键
新建标签页 Ctrl + T
关闭标签页 Ctrl + W
静音当前标签页 Ctrl + M
打开一个新窗口 Ctrl + N
关闭当前窗口 Alt + F4
打开新的 InPrivate 窗口 Ctrl + Shift + N
显示或隐藏收藏夹栏 Ctrl + Shift + B
将焦点放在收藏夹栏中的第一项上 Ctrl + B
将当前选项卡另存为收藏夹 Ctrl + D
在新文件夹中将所有打开的标签页另存为收藏夹 Ctrl + Shift + D
激活地址栏中的 URL Alt + D
打开 “设置及更多...” 菜单 Alt + E 或 F
在页面上查找 Ctrl + F
打开历史记录 Ctrl + H
打开收藏夹管理 Ctrl + Shift + O
打印当前页 Ctrl + P
打开新标签页并切换到该标签页 Ctrl + Shift + T
将焦点放在工具栏中的第一项上 Alt + Shift + T
查看源 Ctrl + U
开始或停止“大声朗读” Ctrl + Shift + U
不带格式粘贴 Ctrl + Shift + V
关闭当前窗口 Ctrl + Shift + W
打开Edge集锦 Ctrl + Shift + Y
放大 Ctrl + 加号 (+)
缩小 Ctrl + 减号 (-)
重置缩放级别 Ctrl + 0（零）
切换到特定标签页 Ctrl + 1、2、3 … 8
切换到最后一个标签页 Ctrl + 9
切换到下一个标签页 Ctrl + Tab
切换到上一个标签页 Ctrl + Shift + Tab
返回 Alt + 向 ← 左键
前进 Alt + 向 → 右键
在当前标签页中打开主页 Alt + Home
